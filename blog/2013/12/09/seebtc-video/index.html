SeeBTC Video

Dec 9th, 2013

Here is a look at my second hackathon project, seebtc. Jorge, Jullian and I ended up winning a prize from one of the API sponsors, ChartIQ. The ChartIQ guy was impressed with how deep we got into the ChartIQ code to enable live streaming of our own data. We are using a Node.js server with an Express framework. We are using web sockets and socket.io to connect to two APIs, MtGox and Blockchain. From MtGox we was recieving every bitcoin trade from thier site, and from Blockchain we are recieving every bitcoin transaction that takes place from thier site. We are saving the trade data to a MongoDB database using the Mongoose ORM and creating candlestick bars every minute with the trade data. We only have 1 view, a dashboard showing the ChartIQ graph with updates every minute, tweets about bitcoins using StockTwits, and a map of the world showing where people are spending bitcoins. Most of this is covered in the video. If you have any questions, feel free to reach out.

Cheers, Kyle

P.S. We currently have this up on cbtc.herokuapp.com but will be running out of space in our MongoDB database soon and will either have to clear the database or take the site down.
